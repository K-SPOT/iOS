# K-SPOT
**개발 기간 : 2018년 8월 31일  ~ 2018년 9월 29일**


## [ About ]

K-SPOT은 **연예인 관련 장소 추천 서비스**입니다. 

* 메인 페이지

  추천 / 인기 장소를 볼 수 있습니다. 

* 카테고리 페이지

  연예인 / 방송별로 관련된 장소를 볼 수 있습니다.

* 장소 페이지

   장소를 선택하거나 내 위치를 지정해 관련된 추천 장소 정보 및 리뷰를 볼 수 있습니다. 필터기능 제공. (set 사용)

* 마이 페이지

  내가 스크랩 / 구독 한 글을 확인 할 수 있습니다.
